The castafiore framework

Welcome.

Castafiore framework is a new web framework with the following features

  1. Write in one single language java
  2. Fully object oriented
  3. No custom compilation to generate javascript like GWT
  4. Can be server centric, or client centric or both. Everything with the same API
  5. Although for backend application there is no need for HTML or CSS, templates can be used easily. Actually, the framework provides the easiest way to use html template. Scripting can be done in any scripting language available with java
  6. Lazy loading of resources like javascript, and css
  7. Very simple API.
  8. Easily integrate any javascript libraries
  9. many more....

Installation

The framework is consisted of the following files

Maven
<dependency>
  <groupId>org.castafiore</groupId>
  <artifactId>org.castafiore.ui</artifactId>
  <version>x.x.x</version>
  <type>war</type>
</dependency>

Download war
http://www..com

Getting started

The easiest way to start coding your first application is probably using maven. You simply need to add the dependency stated above in your pom.xml and your are ready.

If you are not using maven, you can download the war, unzip it, put all the libraries in your class path. Then rename the war into your servlet container. To run your codes, simple deploy all class files in the classes directory of the downloaded war

Writing your first application.

Obviously we are going to start with the good old "Hello world".

After going through this first tutorial, you will be able to do the following

  1. Create a castafiore application
  2. Register the application in the context
  3. Write the necessary code to display "Hello world" on your browser
  4. Deploy and run the application
